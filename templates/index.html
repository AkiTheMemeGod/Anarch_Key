How to Use AnarchKey

Get started in minutes with our Python or Node.js client libraries

1 Install the Package

Install AnarchKeyClient via pip:

pip install AnarchKeyClient

2 Create Account

Sign up for a free account:

Registration

Visit anarchkey.pythonanywhere.com to create your account and get your credentials.

3 Initialize Client

Set up your local authentication token:

anarchkey init --username <YourUsername> --password <YourPassword>

This creates a secure token file at ~/.anarchkey with 0o600 permissions

4 Use in Your Code

Retrieve your API keys securely:

from AnarchKeyClient import AnarchKeyClient

# Initialize the client
client = AnarchKeyClient(
    username="YourUsername",
    api_key="YourAnarchKeyAPIKey"
)

# Retrieve an API key for your project
response = client.get_api_key(project_name="YourProjectName")

# Check if request was successful
if response["success"]:
    api_key = response["key"]
    print(f"Retrieved API key: {api_key}")
else:
    print(f"Error: {response['message']}")

4 Use in Your Code

Retrieve your API keys securely:

const AnarchKeyClient = require('anarchkey-client');

(async () => {
  // Initialize the client
  const client = new AnarchKeyClient({
    apiKey: 'myapikey',
    username: 'me'
  });

  // Retrieve an API key
  const result = await client.getApiKey('myproject');

  if (result instanceof Error) {
    console.error(result.message);
  } else {
    console.log('result:', result);
  }
})();

Programmatic Initialization

You can also initialize programmatically (equivalent to CLI init):

const AnarchKeyClient = require('anarchkey-client');

(async () => {
  try {
    // Initialize and get token
    const token = await AnarchKeyClient.doInit({
      username: 'me',
      password: 'secret'
    });
    console.log('init token:', token);
  } catch (e) {
    console.error('init failed:', e);
  }
})();
